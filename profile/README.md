**e9n.dev** is a curated GitHub organization for maintained projects and autonomous AI agent
workflows.

Unlike my [personal account](https://github.com/espennilsen) which contains experiments and
work-in-progress, this organization hosts only:

## AI-Native Development Experiment

This organization serves as a testbed for **autonomous AI agent development**:

- 🤖 **Multi-agent systems** — Specialized agents collaborating on tasks
- 🔄 **Automated workflows** — AI agents handling development pipelines
- 📋 **Agent coordination** — A2A protocol, task orchestration, handoffs
- 🧪 **Development patterns** — Exploring AI-native software engineering

**Goal:** Demonstrate how autonomous agents can build, maintain, and evolve software projects
with minimal human intervention.
